
🚀 프로젝트 개요
이 프로젝트는 **무인수상선(USV)**의 충돌 회피를 심층 강화학습(DQN) 기반으로 구현한 시뮬레이터입니다.
Otter USV 모델을 사용하여, 정적 장애물과 동적 장애물에 대해 충돌 회피 및 경로 추종을 수행하는 강화학습 환경을 구성합니다.
학습된 에이전트는 Head-on, Crossing, Overtaking, Static Obstacle, Dynamic Obstacle 등 다양한 상황에서 충돌을 회피할 수 있도록 훈련됩니다.

⚙️ 주요 기능
환경 구성:

동적 장애물: 직선 운동, 랜덤 속도

정적 장애물: 고정 위치

자선: Otter USV 모델을 사용한 동역학 시뮬레이션 (속도, 회전, 침로 등)

보상 함수: 경로 추종 보상, 충돌 회피 성공 보상, 충돌 발생 시 음의 보상

탐색 전략: ε-greedy 탐색을 사용하여 충돌 회피 시점을 학습

강화학습 알고리즘: Deep Q-Network (DQN), Dueling DQN

학습 환경:

다양한 조우상황: 단일선박 회피, 다수선박 회피

정적/동적 장애물 복합 시나리오: 다양한 환경 하중, 장애물 종류에 따라 회피 행동을 학습

학습/평가:

Greedy 정책을 통해 학습된 최적 행동을 평가할 수 있는 시뮬레이션 환경
